# Ourizon Web App (Prototype)

React prototype for **Ourizon**, a budget tracking tool. This build implements the **Assets** screen only — other navigation items are visible but inactive.

Based on the design prototype in [`/prototype`](../prototype).

## Run locally

```bash
npm install
npm run dev
```

## Features (Assets)

- Net worth chart with historical data
- Asset groups (Cash, Stocks, Crypto, Super, etc.)
- Edit account balances inline
- Add custom assets via dialog
- Delete user-added assets

## Scope

| Screen   | Status    |
|----------|-----------|
| Home     | Inactive  |
| Assets   | Active    |
| Log      | Inactive  |
| Forecast | Inactive  |
| Family   | Inactive  |
